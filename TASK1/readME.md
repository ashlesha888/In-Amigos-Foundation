# 📢 InAmigos Foundation - Awareness Webpage

A clean, modern, responsive webpage designed to spread awareness about the social impact initiatives and community projects driven by the **InAmigos Foundation**. 

This project was built as part of the Task 1 Internship requirement.

## 🎯 Project Features
*   **Hero Section:** Features a bold value proposition statement with an action-focused entry button.
*   **About Section:** A concise overview explaining the mission, vision, and core values of the NGO.
*   **Projects Showcase:** Highlights key thematic pillars (such as Sustainability) using structural CSS card layouts.
*   **Social Impact Counters:** Uses distinct data visualization milestones (`5,000+ Lives Impacted`) to quickly convey real-world results.
*   **Call-to-Action (CTA):** Uses high-contrast CSS gradients to draw the user's eye directly to structural volunteer signup routes.

## 💻 Tech Stack Used
*   **HTML5:** Structured semantic layout elements (`<header>`, `<section>`, `<footer>`).
*   **CSS3:** Includes responsive layout design via **CSS Grid/Flexbox**, clean typography integration (Google Fonts Poppins), and interactive hover transformations.
